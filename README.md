\# System Communication Dashboard — Django Version



\## Overview

This is a full-stack dashboard built for an internship assignment to log system communications:

\- Emails

\- SMS

\- WhatsApp



The app "does not send real Email/SMS/WhatsApp". It only creates database entries and displays them in tab-based list views.



\## What’s Included

\- Tab UI: Emails / SMS / WhatsApp

\- List views with different columns per tab

\- Modal forms to create logs (AJAX)

\- Dashboard counts (Emails/SMS/WhatsApp)

\- Search per tab

\- Delete with confirmation

\- Export CSV

\- Light/Dark mode toggle (Bootswatch theme switch)



\## Tech Stack

\- \*\*Backend:\*\* Python, Django

\- \*\*Database:\*\* SQLite (default Django DB)

\- \*\*Frontend:\*\* Django Templates + Bootstrap (Bootswatch) + Vanilla JavaScript (Fetch API)
